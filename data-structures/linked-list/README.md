# Linked List

## Methods

### Add

`add(element: any)`

Add an element to end of linked list

### Insert At

`insertAt(element: any, index: number)`

Add an element at a specific index

### Remove From

`removeFrom(index: number)`

Remove node at specific index

### Remove Element

`removeElement(element: any): Node`

Find and remove a specific element

### Index Of

`indexOf(element: any): number`

Return the index of a specific element
